# TurboNetProxy (TurboProxy)

Proxy simples em Python com **menu de gerenciamento via systemd** para abrir/fechar/reiniciar/monitorar portas de forma prática.

✅ Cria serviços no formato: `turboproxy-<PORTA>`  
✅ Instala em: `/opt/NTProxy/`  
✅ Comando do menu: `turboproxy`  
✅ Compatível com Ubuntu/Debian (systemd)

---
